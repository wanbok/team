# Example: iOS App Team

A SwiftUI + TCA (The Composable Architecture) app with backend API.

## Team Composition

| # | Role | Name | Skills |
|---|------|------|--------|
| 1 | PM + Analyst | pm | brainstorming, writing-plans |
| 2 | UI/UX Designer | designer | ui-ux-pro-max |
| 3 | iOS Dev A | ios-a | test-driven-development, composable-architecture |
| 4 | iOS Dev B | ios-b | test-driven-development, composable-architecture |
| 5 | QA Engineer | qa | verification-before-completion |
| 6 | Oracle | oracle | ask-oracle (Sonnet) |

## File Ownership

| Role | Directories |
|------|-------------|
| ios-a | `Sources/FeatureA/`, `Tests/FeatureATests/` |
| ios-b | `Sources/FeatureB/`, `Tests/FeatureBTests/` |
| designer | `docs/design/` |
| qa | `Tests/IntegrationTests/`, `Tests/E2ETests/` |
| pm | `docs/plans/`, `docs/requirements/` |

## Launch Command

```
/team SwiftUI + TCA iOS app for [your app description]
```
